## Using the Script

* Install [node.js](https://nodejs.org/en/)
* open a console / terminal 
* Execute `npm install -g typescript ts-node`
* Execute `npx @wattbewerb/wattbewerb [Gemeindeschlüssel] [Einwohnerzahl]`
